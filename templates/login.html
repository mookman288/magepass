Getting Started

First time using MagePass? MagePass can manage your passwords locally. For more information and an in-depth guide, visit the official documentation.

Local Storage

If you want to store your passwords locally using the default configuration, simply click Create New Account in the menu.

Please note that if you store your passwords locally, you won't be able to access them cross-platform unless you setup a remote solution.
